# SukiWindow

In SukiUI, `SukiWindow` replaces `Window` as the basis for building apps.

## Show

<img src="/controls/layout/sukiwindow.webp" />

## Example

```xml
<suki:SukiWindow>
    <suki:SukiWindow.LogoContent>
        <!-- Logo -->
    </suki:SukiWindow.LogoContent>

    <suki:SukiWindow.MenuItems>
        <!-- Menu -->
    </suki:SukiWindow.MenuItems>
</suki:SukiWindow>
```
