# SukiWindow

在 SukiUI 中, `SukiWindow` 代替 `Window` 成为构建 App 的基础

## 展示

<img src="/controls/layout/sukiwindow.webp" />

## 示例

```xml
<suki:SukiWindow>
    <suki:SukiWindow.LogoContent>
        <!-- Logo -->
    </suki:SukiWindow.LogoContent>

    <suki:SukiWindow.MenuItems>
        <!-- Menu -->
    </suki:SukiWindow.MenuItems>
</suki:SukiWindow>
```
